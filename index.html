盤面轉換工具

使用說明

  1. 設定盤面大小：
    • 輸入行數(rows)和列數(columns)
    • 預設為 3x3 的盤面
  2. 選擇盤面類型：
    • Client 盤面：數字由上到下、由左到右排列
    • Server 盤面：數字由左到右、由上到下排列
  3. 輸入方式（二選一）：
    • 在下方文字框輸入數字陣列（用逗號分隔）
    • 直接點擊「轉換」按鈕，會出現可編輯的空盤面
    • 在盤面格子中直接輸入數字
  4. 美化功能（選用）：
    • 勾選「套用美化」可以將數字轉換為符號
    • 可以自訂符號對應的數字
  5. 轉換結果：
    • 上方顯示輸入的盤面
    • 下方顯示轉換後的數字陣列
    • 右上角紅色數字為索引位置
{ S1: 0, S2: 1, S3: 2, A: 3, K: 4, Q: 5, J: 6, SC: 7, WD: 8, BG: 9, EMPTY: 10 }
